    \textbf{ABC Generators}

    \begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \emph{Correctly Specified Prior} : The 1st ABC Generator is defined as
  \(Y = m*X + c + e\) where \(m \sim N(1,1)\) , \(c=0.5\) and
  \(e \sim N(0,1)\)
\item
  \emph{Misspecified Prior} : The 2nd ABC Generator is defined as
  \(Y = 1 + m*X + c + e\) where \(m \sim N(1,1)\) , \(c=0.5\) and
  \(e \sim N(0,1)\)
\end{enumerate}
